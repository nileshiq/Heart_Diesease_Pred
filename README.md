## 📂 Dataset Overview

The dataset used in this project is focused on predicting the presence of heart disease based on patient health and medical attributes. It contains clinical and physiological data collected from individuals, with all personal identities kept anonymous.

This dataset is widely used for machine learning and healthcare analysis, helping in early detection and prevention of heart related diseases.

The dataset consists of 14 columns, including one target variable:
- patient_id: Unique identifier for each patient  
- heart_disease_present: Target variable (0 = No disease, 1 = Disease present)  

The remaining features include important health indicators such as:
- Age and sex of the patient  
- Chest pain type  
- Resting blood pressure  
- Serum cholesterol levels  
- Fasting blood sugar  
- Resting ECG results  
- Maximum heart rate achieved  
- Exercise induced angina  
- ST depression and slope of peak exercise  
- Number of major vessels  
- Thalassemia test results  

The dataset contains a mix of numerical, categorical, and binary variables, making it suitable for both exploratory data analysis and predictive modeling.

There are no missing values in the dataset, ensuring clean and reliable analysis.
